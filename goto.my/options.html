Other settings

Default search is what we fall back on if nothing matches what was typed. %s is replaced with your query prior to navigation. For example, if I typed `g bar`, my default search was `foo/%s`, and nothing matched `bar`, I would be nagivated to `foo/bar`.

If this field is blank, default search will be disabled
